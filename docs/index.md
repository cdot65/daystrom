## Quick Migration

```bash
npm uninstall -g @cdot65/daystrom
npm install -g @cdot65/prisma-airs-cli
cp -r ~/.daystrom/ ~/.prisma-airs/
```

The CLI binary is now `airs` (was `daystrom`).
